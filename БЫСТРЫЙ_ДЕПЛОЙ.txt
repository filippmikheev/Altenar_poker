ШАГ 1: ОТКРОЙТЕ RAILWAY
═══════════════════════════════════════════════════════════

1. Зайдите на: https://railway.app
2. Нажмите "Login" (в правом верхнем углу)
3. Выберите "Login with GitHub"
4. Разрешите доступ Railway к вашему GitHub

═══════════════════════════════════════════════════════════

ШАГ 2: СОЗДАЙТЕ ПРОЕКТ
═══════════════════════════════════════════════════════════

1. Нажмите "New Project" (синяя кнопка)
2. Выберите "Deploy from GitHub repo"
3. Найдите репозиторий "Altenar_poker"
4. Нажмите на него

Railway автоматически:
  ✅ Определит Node.js проект
  ✅ Установит зависимости
  ✅ Запустит сервер
  ✅ Создаст домен

Ждите 2-3 минуты...

═══════════════════════════════════════════════════════════

ШАГ 3: ПОЛУЧИТЕ URL
═══════════════════════════════════════════════════════════

1. После деплоя нажмите на ваш проект
2. Откройте вкладку "Settings" (шестеренка)
3. Найдите раздел "Domains"
4. Нажмите "Generate Domain" (если нужно)
5. Скопируйте URL

Пример URL:
  https://altenar-poker-production.up.railway.app

✅ ЭТО ВАШ ПОСТОЯННЫЙ URL!

Отправьте его друзьям - они смогут играть! 🎰🎉

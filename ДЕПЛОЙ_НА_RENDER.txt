Render.com - лучший бесплатный хостинг для покера!

═══════════════════════════════════════════════════════════

ШАГ 1: РЕГИСТРАЦИЯ (1 минута)
═══════════════════════════════════════════════════════════

1. Зайдите на: https://render.com
2. Нажмите "Get Started for Free"
3. Выберите "Sign up with GitHub"
4. Разрешите доступ Render к GitHub

═══════════════════════════════════════════════════════════

ШАГ 2: СОЗДАНИЕ СЕРВИСА (2 минуты)
═══════════════════════════════════════════════════════════

1. Нажмите "New +" (вверху справа)
2. Выберите "Web Service"
3. Нажмите "Connect account" (если нужно)
4. Найдите ваш репозиторий "Altenar_poker"
5. Нажмите "Connect"

═══════════════════════════════════════════════════════════

ШАГ 3: НАСТРОЙКА (1 минута)
═══════════════════════════════════════════════════════════

Заполните форму:

  Name: web-poker
  (или любое другое имя)

  Region: Frankfurt (или ближайший к вам)

  Branch: main
  (или master, если у вас master)

  Root Directory: (оставьте пустым)

  Environment: Node

  Build Command: npm install

  Start Command: node server.js

  Plan: Free
  (бесплатный план)

═══════════════════════════════════════════════════════════

ШАГ 4: ДЕПЛОЙ (2-3 минуты)
═══════════════════════════════════════════════════════════

1. Нажмите "Create Web Service"
2. Render начнет деплой
3. Ждите 2-3 минуты
4. Смотрите логи в реальном времени

═══════════════════════════════════════════════════════════

ШАГ 5: ПОЛУЧИТЕ URL
═══════════════════════════════════════════════════════════

После деплоя:

1. Вверху страницы будет ваш URL
   Пример: https://web-poker.onrender.com

2. Скопируйте этот URL

✅ ЭТО ВАШ ПОСТОЯННЫЙ URL!

Отправьте его друзьям - они смогут играть! 🎰🎉

═══════════════════════════════════════════════════════════

✅ ГОТОВО!
═══════════════════════════════════════════════════════════

Теперь ваш покер:
  ✅ Работает на Render.com
  ✅ Доступен по постоянному URL
  ✅ С автоматическим HTTPS
  ✅ Бесплатно!

═══════════════════════════════════════════════════════════

⚠️ ВАЖНО: БЕСПЛАТНЫЙ ПЛАН
═══════════════════════════════════════════════════════════

На бесплатном плане:
  • Сервис "засыпает" после 15 минут бездействия
  • Первый запрос после "сна" занимает ~30 секунд
  • После этого работает нормально

Чтобы сервис не "засыпал":
  • Используйте бесплатный ping сервис (UptimeRobot)
  • Или перейдите на платный план ($7/месяц)

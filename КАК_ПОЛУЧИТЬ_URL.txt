  🌐 КАК ПОЛУЧИТЬ ПУБЛИЧНЫЙ URL
═══════════════════════════════════════════════════════════

📋 БЫСТРЫЙ СПОСОБ:

1. Откройте: http://127.0.0.1:4040
   (Веб-интерфейс ngrok уже открыт)

2. Найдите строку "Forwarding"
   Вы увидите что-то вроде:
   https://xxx.ngrok-free.dev -> http://localhost:3000

3. Скопируйте URL (часть до стрелки):
   https://xxx.ngrok-free.dev

4. Отправьте друзьям!

═══════════════════════════════════════════════════════════

⚠️ ПРОВЕРЬТЕ:

✅ Сервер запущен?
   - Должно быть окно "npm start"
   - Должно быть сообщение "Poker server running on port 3000"

✅ Туннель ngrok запущен?
   - Должно быть окно "ngrok.exe"
   - Должно быть "Session Status: online"

✅ Оба окна открыты?
   - Не закрывайте их, пока играете!

═══════════════════════════════════════════════════════════

🔄 ЕСЛИ URL НЕ РАБОТАЕТ:

1. Проверьте, что сервер запущен:
   - Откройте: http://localhost:3000
   - Должна открыться страница покера

2. Если сервер не работает:
   - Запустите: npm start
   - Или: launcher.bat

3. Перезапустите туннель:
   - Закройте окно ngrok
   - Запустите: start-tunnel.bat
   - Или через launcher: http://localhost:3000/launcher

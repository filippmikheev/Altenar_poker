📋 ВАРИАНТ 1: ПРОСТОЙ (РЕКОМЕНДУЕТСЯ) ⭐

1. Запустите: start-simple-public.bat
2. Подождите 10 секунд
3. Скопируйте URL из окна "Cloudflare Tunnel"
4. Отправьте друзьям!

✅ Не требует регистрации
✅ Работает сразу
✅ Не нужен Docker

═══════════════════════════════════════════════════════════

📋 ВАРИАНТ 2: ЧЕРЕЗ DOCKER (если Docker Desktop запущен)

1. Запустите Docker Desktop
2. Запустите: start-cloudflare.bat
3. Подождите 15 секунд
4. URL будет в файле: PUBLIC_URL.txt
   или в логах: docker-compose logs cloudflare-tunnel

✅ Автоматический запуск
✅ Все в контейнерах

═══════════════════════════════════════════════════════════

📋 ВАРИАНТ 3: NGROK (если есть токен)

1. Запустите: start-tunnel.bat
2. Откройте: http://127.0.0.1:4040
3. Скопируйте URL из строки "Forwarding"
4. Отправьте друзьям!

✅ Стабильный URL
⚠️ Требует регистрации и токена

═══════════════════════════════════════════════════════════

📋 ВАРИАНТ 4: LOCALNETWORK (только в одной сети)

1. Узнайте ваш IP: ipconfig
2. Найдите IPv4 (например: 192.168.1.100)
3. Отправьте друзьям: http://ВАШ_IP:3000

✅ Работает без интернета
⚠️ Только в одной сети (Wi-Fi/LAN)

═══════════════════════════════════════════════════════════

💡 РЕКОМЕНДАЦИЯ:

Используйте ВАРИАНТ 1 (start-simple-public.bat) - 
самый простой и надежный!

═══════════════════════════════════════════════════════════

⚠️ ВАЖНО:

✅ Не закрывайте окно туннеля
✅ URL работает, пока запущен туннель
